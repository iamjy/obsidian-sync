# AI 시대 코드 리뷰 체크 리스트

본 문서는 최신 AI 프로그램 및 시스템 개발 관련 코드 리뷰에서 중요하게 고려해야 할 사항들을 정리한 참고 자료입니다. 다음은 각 검토 항목별로 구체적으로 확인해야 할 부분들을 설명합니다.

## 리뷰 체크 리스트

1. AI 가정 및 입력 데이터 검증
   - AI 모델이 사용하는 전제와 가정의 타당성을 체크하고, 예상치 못한 입력 시나리오를 모색합니다.
1. 엣지 케이스 커버리지 확인
   - 비일반적이거나 예기치 않은 사용 사례 및 데이터 입력에 대해 코드의 안정성을 검토하여, 잠재적 버그를 최소화합니다.
3. 비즈니스 로직 정확성 확인
   - 시스템이 기업의 비즈니스 규칙 및 요구사항을 충족할 수 있도록, 올바른 논리 구조와 데이터 처리가 이루어지는지 확인합니다.
4. 기술 부채 평가
   - 장기적인 운영과 유지보수성을 위해, 즉각적인 수정이 아닌 근본적 개선 필요 여부를 판단하고 관리 계획을 수립합니다.
5. 유지 보수 및 확장성 고려
   - 코드의 구조와 설계가 변경, 추가 기능 도입 시에도 반응적이며, 향후 발전 가능성 있는 아키텍처를 지니고 있음을 평가합니다.
