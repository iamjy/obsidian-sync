### **2. Light Model 옵션별 비교**  
#### **(A) Light Model**  
- **핵심 특징:**  
  - 영상 **단방향 전달** (ACS는 SW/알고리즘 수행 불가)  
  - 제이시스메디칼의 **제안 목적과 일치**  
- **결론:** 단순 촬영 영상 기반 제공 (추가 개발 불필요)  

#### **(B) Light Model**  
- **핵심 특징:**  
  - **영상 정렬 기능 포함** (영상 촬영 + 정렬)  
  - **Standard HW SW와 별도 개발 필요** (HW 차이로 인한 SW 신규 개발)  
- **결론:** 기존과 상이한 아키텍처로 별도 R&D 투자 필수  

#### **(C) Standard Model**  
- **현황:**  
  - 기존 사양 기반 검토 진행 중  
  - **AI 제외, Geometry 기능 개발 가능성 미확인** (제조사 자료 확보 필요)  

---

### **3. 즉시 확인이 필요한 주요 포인트**  
1. **SOM Geometry 기능 가능성 확인**  
   - AI 제외, 제조사 문서로 기술 가능성 검증 필요  

2. **IPAD 기기 활용 방식 명확화**  
   - 영상 전달 방식 (단순 촬영 영상 vs. RGB + Thermal 합성)  
   - **헤드 오리엔테이션 연산 가능성** (다른 앱 참고 시 가능 예상)  

3. **얼굴 Tracking 사양 정의**  
   - **실시간 X**  
   - 주기적 헤드 오리엔테이션 분류 (정면/ 좌측/ 우측 등)
